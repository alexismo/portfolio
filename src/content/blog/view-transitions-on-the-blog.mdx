## Smooth navigation between list and detail

View transitions make moving from the blog index to a post feel continuous instead of abrupt.
Naming key elements—like the post title and hero area—lets the browser animate them across route changes.

### Naming elements for continuity

The blog header uses a dedicated `view-transition-name` so it can morph in place while the rest of the page cross-fades.
That small detail keeps context visible during navigation.

```css
.blog-header {
  view-transition-name: --blog-header;
  font-size: 6.4rem;
}
```

```javascript
document.addEventListener("astro:after-swap", () => {
  const header = document.querySelector(".blog-header");
  if (header) {
    header.style.viewTransitionName = "--blog-header";
  }
});
```
